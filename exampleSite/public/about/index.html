About

This is a demo site showcasing the Flavor Hugo theme.

The Theme

Flavor is a minimal Hugo theme with Tailwind CSS and Zettelkasten-inspired knowledge management features. It’s designed for:

  • Personal blogs and websites
  • Digital gardens and knowledge bases
  • Technical documentation
  • Online portfolios

Features Demonstrated

This demo showcases:

Zettelkasten Features

  • Wikilinks: [[double bracket]] syntax and shortcodes
  • Backlinks: Automatic detection of linking pages
  • Knowledge Graph: Interactive D3.js visualization
  • Note Types: Fleeting, literature, permanent, and MOC
  • Related Notes: Algorithmic suggestions

Content Features

  • Articles: Long-form technical content
  • Notes: Atomic knowledge units
  • Projects: Portfolio showcase
  • Search: Fuzzy full-text search

Design Features

  • Dark Mode: Automatic or manual
  • Responsive: Mobile-friendly
  • Accessible: Semantic HTML, ARIA labels
  • Fast: Static generation, minimal JavaScript

Get Started

  1. Clone the theme: git clone https://github.com/a-kostevski/flavor
  2. Copy the example site
  3. Customize hugo.toml
  4. Start writing!
