# Flavor

A minimal Hugo theme with Tailwind CSS.

## Installation

Add this theme as a git submodule:

```bash
git submodule add https://github.com/a-kostevski/flavor.git themes/flavor
```

## Configuration

Set the theme in your `hugo.toml`:

```toml
theme = "flavor"
```

## Development

Install dependencies:

```bash
npm install
```

Build CSS:

```bash
npm run build
```

Watch for changes:

```bash
npm run dev
```

## Features

- Tailwind CSS with typography plugin
- Dark mode support
- Responsive design
- Custom shortcodes (admonition, sidenote, spoiler, wikilink)
- Backlinks support
- Search functionality
- Projects section layout
